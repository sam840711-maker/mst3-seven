# MST3 세븐스플릿 (v0.1) — 독립 슬리브 앱

SH Portfolio · 리밸런서와 인프라를 공유하는 sibling PWA. MST3(3x MSTR)의 변동성을
세븐스플릿으로 수확하는 **완전 격리 슬리브**를 운용한다.

## 확정 설정 (공격형, 워크포워드 검증)
- 슬리브 하드캡 3,000만원 · 분할 10 · 진입 −6% · 트란셰 익절 +8%
- 20일 MA 추세필터 · 실현익 스윕 · 무레버리지·무신용
- 검증: OOS(2024~2026) 총 2.28배, 연 39%, 최대낙폭 −37%. 전체(5.9년) 연 22%.
  → 낙폭 −37%(≈−1,100만원) 구간을 규칙대로 버틸 것.

## 매일 운용
1. "시세 불러오기"(또는 현재가 수동 입력 → 현재가 적용).
2. 히어로 **오늘의 신호** 확인: 매수 / 익절 / 관망.
3. 실제 체결 후 "매수 기록" 또는 "익절 기록"으로 상태 갱신.
4. 트란셰 사다리·실현익·여력 확인.

## 배포
- GitHub Pages 리포의 하위 폴더(예: `/mst3/`)에 7파일 업로드.
- OAuth 클라이언트 ID는 sibling 앱과 동일(도메인 이미 승인됨). 스코프는
  `drive.appdata`(앱 전용 폴더, 다른 앱 데이터와 격리).
- 시세는 `sh-proxy`로 Yahoo MST3.L 조회. 프록시 형식이 다르면 설정에서 조정하거나
  현재가 수동 입력(신호는 정상 작동).
